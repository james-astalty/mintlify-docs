## Overview

Support Worker Case Note Templates let you standardise how case notes are created and recorded across your organisation. By setting up a template, you can guide Support Workers on what information to include, ensure consistency in reporting, and save time by pre-filling common sections. Templates can be customised with headings, prompts, or instructions, and once published, they appear in the Astalty application for Support Workers to use when writing their case notes.

## Support Worker Case Note Templates

Similar to other note templates, you can create a Support Worker case note template to standardise how notes are recorded.

To create one:

- Go to Settings \> [**Tasks and Notes**](https://app.astalty.com.au/settings/tasks-notes).
- Select Create Note Template.
- Enter a Note Template Name.
- Set the Usage Type to Support Notes.
- Add any further details or instructions.

![image2-1757911370.png](/images/image2-1757911370.png)

Once saved, this template will appear in the Astalty application for Support Workers, allowing them to write case notes using your standard format.
