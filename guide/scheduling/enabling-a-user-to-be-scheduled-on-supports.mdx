## Overview

In Astalty, you can control which users are eligible to be scheduled on supports. This can be set either when inviting a new user or by updating an existing user’s profile.

## Method 1: Invite a New User as a Support Worker

- From the left-hand side menu, go to [**Users**](https://app.astalty.com.au/users').
- Click Invite User.
- Select Support Worker as the role.
- Tick This user can be scheduled for support.

![image2-1757910955.png](/images/image2-1757910955.png)

- Fill in the required information.
- Scroll to the bottom of the page and Invite User.

## Method 2: Update an Existing User

- From the left-hand side menu, go to [**Users**](https://app.astalty.com.au/users).
- Click on the user’s Profile.
- On the right-hand side, select Edit.
- Tick This user can be scheduled for support.

![image1-1757910978.png](/images/image1-1757910978.png)

- Scroll to the bottom of the page and click Update User.
