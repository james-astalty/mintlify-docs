## Overview

You will see a 'more' menu option within the participant menu. This section currently hosts the;

- Participant Audit Trail

![More Participants](/images/more-participants.png)

## Participant Audit Trail

You can now view a participant audit trail for all updates to the participants profile via this menu!

![Audit Trail](/images/audit-trail.png)

Simply select 'view audit trail' to view the audit trail for this participants profile.

![Audit Trail View](/images/audit-trail-view.png)
