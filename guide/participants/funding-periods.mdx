## **Overview**

Astalty supports adding services for an NDIS Plan that has funding periods associated with it. For more information about funding periods, please [**visit the NDIS website**](https://ndis.gov.au/changes-ndis-legislation/frequently-asked-questions-about-legislation#funding). This feature is available for all funding types this includes NDIA, Plan, and Self Managed participants.
